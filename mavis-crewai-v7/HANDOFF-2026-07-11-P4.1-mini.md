**P4.1 mini = 8 机制各 1 query 完整覆盖 + 修复 4 bug (Tuple + 错方法 + 重建 router + 关键词命中跳 LLM)**

效果:

- ✅ 4 bug 全修

### 4 bug 修复 (永久 invariant #49)

| Bug | 原因 | 修复 |
|---|---|---|
| 1. `Tuple` 未定义 | router.py 缺 import | `from typing import Dict, List, Optional, Tuple` |
| 2. `EightMechRouter.query()` AttributeError | EightMechRouter **无 .query() 方法**, 只有 .route_and_response() | 改用 `route_and_response(q["query"])` |
| 3. 200 query 每次重建 router | 每次 `EightMechRouter(top_k=3)` 重建 8 query_engine | **一次性建, 复用 200 次** (省 0.8s × 200 = 160s) |
| 4. 关键词命中还调 LLM 兜底 | `call_llm_router` 无条件跑 | 关键词命中时跳过 LLM (省 5s/次 × 200 = 17 分钟) |
