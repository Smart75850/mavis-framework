## 复用今晚经验

### HttpxOllamaEmbedding 绕过 ollama lib (P1.3 新增)

- ollama Python lib 装需要 `socksio` 依赖 (httpx 配 SOCKS 代理)
- 直接用 `httpx` 调 Ollama HTTP API 避开依赖
- P1.3 已固化到 `HttpxOllamaEmbedding` class

### SimpleDirectoryReader 默认不识 .md (P1.3 新增)

- `required_exts=[".md"]` 会导致 `_add_files` 报 "No files found"
- 改用 `input_files=[str(f) for f in md_files]` 显式传文件列表

### rglob 需排除 archive/hooks-templates/.bak (P1.3 新增)

- mavis memory 子目录有历史版本 (archive/, hooks-templates/)
- `.bak` 和 `.summary` 是临时文件
- 必须 exclude_dirs + exclude_suffixes 过滤
